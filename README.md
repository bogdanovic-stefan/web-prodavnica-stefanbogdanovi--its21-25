# Mini web prodavnica

## Naziv prodavnice
Restoran Gagi

## Tema prodavnice
Restoran

## Opis projekta
Ovaj repozitorijum sadrži kompletno razvijenu e-commerce web prodavnicu za restoran "Gagi", realizovanu u WordPress/WooCommerce okruženju. Projekat obuhvata ceo korisnički tok – od pregleda menija i dinamičkog dodavanja artikala u korpu, pa sve do funkcionalne stranice za naplatu (Checkout) i praćenja porudžbina preko korisničkog naloga. Ceo sajt je vizuelno prilagođen ugostiteljskoj niši i u potpunosti je optimizovan za responzivan prikaz na svim uređajima.
